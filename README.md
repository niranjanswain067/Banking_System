# Core Java Banking System (JDBC + MySQL)

A clean and simple Banking Management System built using Core Java and JDBC, designed for students and beginners. This project uses a menu-driven console interface and performs all essential banking operations such as creating accounts, updating details, deposit/withdraw, balance check and closing accounts.
